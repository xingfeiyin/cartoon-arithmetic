# Growth Checklist

这个清单用于做真实、合规的项目增长。

## GitHub 仓库

- [ ] README 首页清晰说明项目价值
- [ ] LICENSE 显示为 MIT
- [ ] 添加 README_EN.md
- [ ] 添加 CONTRIBUTING.md
- [ ] 添加 ROADMAP.md
- [ ] 添加算法索引
- [ ] 添加 Issue 模板
- [ ] 添加 PR 模板
- [ ] 添加 GitHub topics

## 内容建设

- [ ] 发布一篇项目介绍文章
- [ ] 发布一篇算法学习文章
- [ ] 把重点算法整理成系列内容
- [ ] 每周至少补充一次文档或测试
- [ ] 把适合新手的任务标记出来

## 申请准备

- [ ] 准备项目重要性说明
- [ ] 准备 API 额度用途说明
- [ ] 准备维护计划
- [ ] 准备近期提交记录
- [ ] 准备真实反馈或文章链接

## 不做的事情

- [ ] 不刷 star
- [ ] 不刷 fork
- [ ] 不制造虚假 issue
- [ ] 不购买虚假流量
