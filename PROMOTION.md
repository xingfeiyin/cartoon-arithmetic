## 掘金 / 知乎

我整理了一个 Java 算法学习项目：cartoon-arithmetic。

这个项目主要把《漫画算法》里的经典问题转成可以直接运行的代码示例，内容包括排序算法、面试题、LRU 缓存、A* 寻路、位图、大整数相加等。

它不是刷题平台，而是一个更适合“看懂思路、跑通代码、自己改一改”的学习仓库。适合刚开始学数据结构与算法的人，也适合准备面试时快速复习经典问题。

项目地址：
https://github.com/xingfeiyin/cartoon-arithmetic

如果你也在学算法，欢迎 star、fork、提 issue 或补充测试用例。
