# cartoon-arithmetic

Runnable Java examples and study notes for classic algorithms, data structures, interview problems, and practical algorithm use cases inspired by Cartoon Algorithm.

This project is designed for learners who want to understand algorithms by reading small, concrete Java programs instead of only reading abstract explanations.

## What Is Included

### Interview Problems

- Big number addition
- Finding missing integers
- Finding the next permutation
- Greatest common divisor
- Linked list cycle detection
- Maximum sorted adjacent distance
- Min stack
- Power of two check
- Remove K digits
- Queue implemented with stacks

### Sorting Algorithms

- Bubble sort
- Cocktail sort
- Quick sort
- Heap sort
- Counting sort
- Bucket sort

### Practical Examples

- A* search
- LRU cache
- Bitmap
- Random red packet allocation

## Why This Project Matters

Many algorithm resources explain ideas well, but beginners often struggle to connect those ideas with runnable code. This repository focuses on that missing step: each example is small enough to read, run, modify, and discuss.

It can be used for self-study, interview review, classroom examples, and beginner-friendly open source contributions.

## How To Run

The project uses Java 8 and Maven.

```bash
./mvnw test
```

You can also open the project in an IDE and run the `main` method in each example class.

## Learning Path

1. Start with sorting examples.
2. Move to interview problems such as big number addition and cycle detection.
3. Study practical cases such as LRU cache and A* search.
4. Add tests or edge cases to confirm your understanding.
5. Submit improvements through issues or pull requests.
