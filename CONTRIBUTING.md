# 贡献指南

感谢你愿意改进 cartoon-arithmetic。这个项目的目标是让算法学习更直观、更容易运行和复习。

## 可以贡献什么

- 修复示例代码中的问题
- 补充测试用例
- 增加题目说明和解题思路
- 改进变量命名和注释
- 增加新的算法示例
- 补充 README 索引
- 翻译或补充英文说明

## 提交建议

1. 每次提交尽量只解决一个问题。
2. 新增算法时，请同时补充题目说明、核心思路和可运行示例。
3. 修改已有示例时，请尽量保留原来的学习意图。
4. 如果修复边界情况，请说明触发场景。
5. 提交前请运行项目，确认没有明显错误。

## 新增示例推荐格式

```java
/**
 * 题目：这里写题目描述
 * 思路：这里写核心思路
 */
public class Example {
    public static void main(String[] args) {
        // 放一个最小可运行示例
    }
}
```

## Issue 建议

如果你只是发现问题但暂时不想提交代码，也欢迎提 Issue。请尽量说明：

- 你看到的问题是什么
- 哪个文件或示例受影响
- 你期望的结果是什么
- 如果有报错，请贴出报错内容

## Pull Request 建议

提交 PR 时可以简单说明：

- 改了什么
- 为什么改
- 如何验证

谢谢你的贡献。
