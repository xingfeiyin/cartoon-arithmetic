# Roadmap

这个路线图用于帮助项目逐步变成更完整的算法学习资料。

## 第一阶段：让项目更容易读

- 补充完整 README
- 给每个示例增加题目说明
- 增加目录索引
- 标注每个算法的难度和适用场景

## 第二阶段：让项目更容易跑

- 补充单元测试
- 增加更多输入样例
- 覆盖常见边界情况
- 整理 Maven 运行说明

## 第三阶段：让项目更适合学习

- 为重点算法补充图解说明
- 增加复杂度说明
- 增加常见错误写法对比
- 增加面试追问整理

## 第四阶段：让项目更容易参与

- 增加 Issue 模板
- 增加贡献指南
- 标注适合新手参与的问题
- 补充中英文说明

## 第五阶段：申请开源支持

- 整理项目价值说明
- 准备 API 额度使用计划
- 记录真实维护节奏
- 收集真实用户反馈
- 发布项目介绍文章

## 候选新增内容

- 二叉树遍历
- 红黑树完整实现
- 哈希表基础实现
- 动态规划经典题
- 图的遍历
- Dijkstra 最短路径
- Bloom Filter
- 跳表
- Trie 树
- 并查集
