# 建议创建的 Issues

这些 Issue 可以帮助项目显得更清晰，也方便真实用户参与。

## 1. 补充单元测试覆盖经典算法示例

当前项目里很多示例可以通过 `main` 方法运行，但缺少系统化测试。建议优先给排序算法、大整数相加、LRU 缓存等示例补充测试。

标签建议：`good first issue`, `test`

## 2. 为 README 增加算法索引表

可以把每个示例整理成表格，包含名称、文件路径、所属分类、学习重点，方便访问者快速找到想看的内容。

标签建议：`documentation`, `good first issue`

## 3. 补充红黑树完整实现

当前仓库里已有红黑树入口文件，但内容还不完整。可以补充插入、旋转、染色调整和基础测试。

标签建议：`enhancement`, `algorithm`

## 4. 给重点示例增加复杂度说明

建议先从排序算法、LRU 缓存、A* 寻路开始，补充时间复杂度、空间复杂度和适用场景。

标签建议：`documentation`

## 5. 增加英文 README

为项目增加英文介绍，帮助非中文用户理解项目内容和使用方式。

标签建议：`documentation`, `help wanted`
